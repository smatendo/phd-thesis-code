# Soil Hydraulic Properties

This folder contains the Python code used for the analysis of soil hydraulic properties in irrigated agricultural fields in Perkerra and Tana, Kenya.

The analyses include:

* processing of field infiltration measurements;
* estimation of field-saturated hydraulic conductivity;
* implementation and evaluation of infiltration models;
* comparison of soil hydraulic properties before and after irrigation;
* statistical analysis and visualisation of the results.

## Software

The analyses were conducted in Python using Jupyter Notebook.

## Data availability

The input datasets are not included in this repository. Data may be made available upon reasonable request, subject to the applicable data-sharing conditions and permissions.

## Recommended execution order

The notebooks should be executed according to the numbering included in their filenames.
