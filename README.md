# PhD Thesis Code

This repository contains selected code developed for the analyses presented in the PhD thesis of Sara Esperanza Matendo.

The research focuses on soil salinity and sodicity, soil hydraulic properties, soil biological functioning, and climate-smart agriculture in irrigated arid and semi-arid environments.

## Repository structure

The code is organised into three studies:

* `01-soil-hydraulic-properties/`: infiltration modelling and estimation of soil hydraulic properties.
* `02-salinity-sodicity-biology/`: statistical analyses of salinity, sodicity, soil biological indicators, and surface hydraulic functioning.
* `03-climate-smart-agriculture/`: statistical analyses of climate-smart agriculture adoption and socio-institutional indicators.

The code developed for the AquaLeach and proximal sensing and machine-learning studies is not included in this repository.

## Data availability

This repository contains source code only. The datasets used in the analyses are not publicly included because of data-sharing restrictions and confidentiality considerations.

Data may be made available upon reasonable request to the author, subject to the applicable data-sharing conditions and permissions.

## Software

All analyses included in this repository were conducted in Python using Jupyter Notebook.

The required Python packages and the expected input-data structure are described within each study folder.
