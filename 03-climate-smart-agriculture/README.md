# Climate-Smart Agriculture Adoption and Gender Gaps

This folder contains the Python code used to analyse climate-smart agriculture and salinity-mitigation practice adoption, socio-institutional factors, and gender-related constraints in the Perkerra and Tana irrigation schemes, Kenya.

The analyses include:

* descriptive analysis of farmer characteristics and climate-stress perceptions;
* Pearson chi-square tests;
* comparison of adoption rates across irrigation schemes and training levels;
* logistic regression of climate-smart agriculture adoption;
* analysis of reported land degradation;
* assessment of gender gaps in access to resources and decision-making;
* logistic regression of crop-marketing participation;
* analysis and visualisation of open-ended survey responses.

## Software

All analyses were conducted in Python using Jupyter Notebook.

## Data availability

The survey dataset is not included because it contains confidential participant information. An anonymised version may be made available upon reasonable request, subject to the applicable data-sharing conditions and permissions.

## Reproducibility

The notebooks should be executed according to the numbering included in their filenames. Users must provide an appropriately anonymised input dataset and preserve the expected variable names described within each notebook.
