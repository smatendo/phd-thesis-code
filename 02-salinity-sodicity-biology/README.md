# Salinity, Sodicity, Biological and Hydraulic Functioning

This folder contains the Python code used for the integrated assessment of soil salinity, sodicity, biological indicators, geochemical gradients, and surface hydraulic functioning in the Perkerra and Tana irrigation schemes, Kenya.

The analyses include:

* descriptive statistical analysis;
* comparison of soil properties between irrigation schemes and soil layers;
* principal component analysis of soil physicochemical and pXRF-derived variables;
* Spearman correlation analysis;
* Type II ANCOVA using HC3 heteroskedasticity-robust covariance estimates;
* ordinary least-squares and robust regression analyses;
* Mann–Whitney U tests and assessment of soil properties associated with preferential flow;
* generation of the tables and figures presented in the thesis.

The notebooks reproduce Figures 4.11–4.15, Tables 4.4–4.5, and the supplementary Figures A.1–A.2 and Tables A.1–A.4.

## Software

All analyses were conducted in Python using Jupyter Notebook.

## Reproducibility

The notebooks should be executed according to the numbering included in their filenames. Users must provide the required input datasets and preserve the expected filenames, worksheet names, and variable names described within each notebook.
