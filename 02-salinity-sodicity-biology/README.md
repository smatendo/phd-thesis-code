# Salinity, Sodicity, Biological and Hydraulic Functioning

This folder contains the Python code used for the integrated assessment of soil salinity, sodicity, biological indicators, and surface hydraulic functioning in the Perkerra and Tana irrigation schemes, Kenya.

The analyses include:

* descriptive statistical analysis;
* comparison of soil properties between irrigation schemes and soil layers;
* principal component analysis;
* Spearman correlation analysis;
* Type II ANCOVA using HC3 heteroskedasticity-robust covariance estimates;
* ordinary least-squares and robust regression analyses;
* assessment of soil properties associated with preferential flow;
* generation of tables and figures presented in the thesis.

## Software

All analyses were conducted in Python using Jupyter Notebook.

## Data availability

The datasets are not included in this repository. Data may be made available upon reasonable request, subject to the applicable data-sharing conditions and permissions.

## Reproducibility

The notebooks should be executed according to the numbering included in their filenames. Users must provide the required input datasets and preserve the expected variable names described within each notebook.
